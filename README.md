# CoreCMS

CoreCMS is a lightweight CMS backend built with PHP following a custom MVC architecture.

The project provides a RESTful API allowing content management through CRUD operations. It communicates with a MySQL database using PDO and returns JSON responses with proper HTTP status codes.

## Features

- Create posts
- Read posts
- Update posts
- Delete posts
- RESTful API
- JSON responses
- Custom router

## Tech stack

- PHP
- MySQL
- PDO
- REST API
- MVC architecture
